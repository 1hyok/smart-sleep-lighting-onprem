# 엣지 노드 (Raspberry Pi) - 스마트 수면 조명 제어 시스템

이 저장소는 '개인화 수면 루틴 조명 관리 서비스'의 **엣지 노드(Raspberry Pi) 환경의 센서 연동, 조명 제어, MQTT 통신**만을 담당합니다.

> ⚠️ **역할 경계 안내**
> 본 모듈은 **엣지 노드(센서/액추에이터/MQTT)** 기능만 수행합니다.
> Fitbit Web API (OAuth 연동), REST/SSE 기반 API 서버, SQLite 데이터베이스 파이프라인 및 로컬 대시보드 구현은 **백엔드/프론트엔드 담당 팀원의 책임 범위**이며, 이 저장소에는 포함되지 않습니다.

## 기술 스택
- **런타임**: Node.js
- **메시지 브로커**: Mosquitto MQTT 브로커 (로컬)
- **하드웨어 제어**:
  - YL-40 (PCF8591 I2C 조도 센서)
  - 4핀 RGB LED (PWM 제어, `pigpio` 활용)
- **저장소(엣지 임시 버퍼)**: JSONL 이벤트 로그 (`service_log.jsonl`)
- **포트**: MQTT `:1883`

## 핵심 아키텍처
1. **센서 연동**: I2C 통신을 통한 조도 데이터 수집 (노이즈 필터링 적용)
2. **조명 제어**: PWM 제어를 통한 취침/기상 스케줄에 따른 자연스러운 점진적 페이딩 (감마 보정 적용)
3. **MQTT 통신**: 백엔드 서버와의 비동기 통신, QoS 1 보장, 재연결 백오프 및 LWT(Last Will & Testament) 상태 관리
