# 엣지 노드 (Raspberry Pi) — 조도 센서 MQTT 발행

이 저장소는 '개인화 수면 루틴 조명 관리 서비스'의 **엣지 노드 전용** 애플리케이션입니다. Raspberry Pi 에 연결된 조도 센서(LDR + PCF8591 I2C ADC)의 값을 수집하여 1차 노이즈 필터링을 거친 뒤, MQTT 브로커로 발행하는 단일 책임만 수행합니다.

> ⚠️ **역할 경계 안내**
> 본 모듈은 **엣지 컴퓨팅(센서 연동 / 노이즈 필터링 / MQTT 발행)** 만 담당합니다.
> 조명 제어, 취침/기상 루틴, 자동 추천, 시간 기반 스케줄링, Fitbit 연동, REST API 서버, SQLite 데이터베이스, 사용자 인증/대시보드 등 **모든 비즈니스 로직과 사용자 대면 표면은 백엔드/프론트엔드 팀의 책임 범위**이며 이 저장소에는 포함되지 않습니다.

## 기술 스택
- **런타임**: Node.js
- **메시지 브로커**: Mosquitto MQTT 브로커 (로컬, `:1883`)
- **센서**: YL-40 보드 (PCF8591 I2C 8-bit ADC + LDR)
- **버스**: I2C (`/dev/i2c-1`, 주소 `0x48`)

## 핵심 동작
1. **센서 수집**: I2C 통신을 통한 조도 ADC 원시값 read
2. **노이즈 필터링**: 버스트 메디안(스파이크 제거) + 이동 평균(평활화) 2단계
3. **MQTT 발행**: `home/sensor/illuminance` 토픽으로 정제된 lux 추정값 주기 발행 (QoS 1)
4. **디바이스 상태**: `home/edge/status` 토픽으로 LWT(Last Will & Testament) 기반 online/offline 자동 보고

발행 페이로드 스펙과 백엔드 연동 가이드는 [INTEGRATION.md](INTEGRATION.md) 참조.
